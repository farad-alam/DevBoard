What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Gets or sets the HTML inside an element, including tags. innerText: Gets or sets the visible text inside an element . textContent: Gets or sets the raw text .

What is event delegation in the DOM?

Event delegation is when don't need add an event listener to each individual element, but instead add it to a parent element. The event is "delegated" from the parent to the child elements.

What is event bubbling in the DOM?

Event bubbling is when an event starts from the target element and then "bubbles up" to its parent elements. So if I click on a button, the click event will also trigger on its parent elements.
